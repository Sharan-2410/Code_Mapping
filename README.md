# Code_Maping Project

A small single-file HTML project containing `Untitled2_html.html` — a simple static page you can open in any browser.

**Contents**

- `Untitled2_html.html`: Main HTML file (single-page demo).

**How to open**

- Double-click `Untitled2_html.html` in File Explorer to open in your default browser.
- Or from PowerShell / Command Prompt run:

```powershell
start "" "Untitled2_html.html"
```

**Usage**

- Edit `Untitled2_html.html` with any text editor (VS Code, Notepad++, or Notepad).
- Save changes and refresh the browser to preview.

**Notes**

- No build tools or dependencies required — static HTML only.
- Rename the file if you want a more descriptive project name.
